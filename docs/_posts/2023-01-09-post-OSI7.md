# OSI 7계층

> 국제표준화기구(ISO)에서 개발한 모델로, 컴퓨터 네트워크 프로토콜 디자인과 통신을 계층으로 나누어 설명한 것
> => 네트워크에서 통신이 일어나는 과정을 7계층으로 나눈 모델

#### 탄생 배경

같은 회사 컴퓨터끼리만 네트워크를 연결할 수 있었고 서로 다른 기종간의 네트워크 필요성이 대두됨
호환성 이슈를 방지하고 모든 시스템들 간의 상호 연결에 문제가 없도록 ISO에서 OSI 참조 모델을 표준으로 제시

> 💡네트워크(Net + Work)
> 통신 장비끼리 마치 그물망처럼 연결되어 데이터들을 주고 받는 집단

#### 7계층으로 나눈 이유

통신이 일어나는 과정을 단계적으로 파악하기 용이하기 때문
각 계층은 독립적인 상하구조의 계층적 특성으로 특정 계층에 이상이 생겼을 때 다른 계층은 확인할 필요없이 문제가 있는 계층만 고쳐서 문제를 해결할 수 있음
※ 상하구조 체계는 윗 계층이 동작하기 위해선 아래 계층들이 잘 작동되어야 함을 전제로 함
배우는 사람들이 쉽게 습득 가능하고, 표준모델이 있기에 의사소통에 용이

#### OSI 7계층과 TCP/IP 5계층

데이터 표준을 정리한 것이 OSI 7계층이었다면, 이론을 실제 사용하는 인터넷 표준은 TCP/IP 5계층
